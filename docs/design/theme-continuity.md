# Theme Continuity

Pluckr v2 starts from the same visual personality as the current Swift prototype.

## Preserved Tokens

- `sage`: `#7FB785`
- `forest`: `#5D7C6B`
- `mint`: `#E8F5E9`
- `cream`: `#FBF8F3`
- `background`: `#FEFCF7`
- `charcoal`: `#2C3E50`

## Preserved Interface Feel

- warm paper-like surfaces instead of flat startup gray
- rounded cards with soft depth instead of hard enterprise panels
- clinical but gentle accenting instead of loud saturated color
- readable, journal-inspired hierarchy instead of cramped dashboard density

## What This Means Practically

- mobile and web both read from `@pluckr/design-system`
- the first v2 screens should feel recognizably Pluckr on sight
- rework can happen later, but drift should not happen by accident
